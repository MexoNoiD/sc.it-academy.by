Блок навигации

  • Ширина 15em
  • Высота определается контентом
  • Должен присутствовать цвет фона
  • Распологается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере. Еще немножно наполнения, чтобы захватывала статью 2, при развернутом на весь экран режиме

Статья 1

Размер блока:

Занимает всю доступную ширину

Высота по размеру контента

Должен присутствовать цвет фона

Во всех "статьях" сделать отступ снизу от соседей равным 2em, при помощи свойства margin-bottom

Статья 2

Размер блока:

Занимает всю доступную ширину

Высота по размеру контента

Должен присутствовать цвет фона

Во всех "статьях" сделать отступ снизу от соседей равным 2em, при помощи свойства margin-bottom. Картинка вставленна при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
